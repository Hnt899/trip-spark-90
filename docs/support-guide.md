# Руководство по чат-помощнику TudaSuda

## 1. Назначение

Чат-помощник отвечает на типовые вопросы пользователей по поиску, покупке и обслуживанию билетов.

## 2. Как открыть чат

1. Откройте сайт TudaSuda.
2. Нажмите кнопку/виджет чата в интерфейсе.
3. Введите вопрос и отправьте сообщение.

[СКРИНШОТ: открытый чат-помощник]

## 3. Какие вопросы обрабатываются автоматически

- поиск рейса и выбор направления;
- этапы оформления и оплаты;
- работа личного кабинета;
- возврат/обмен билетов;
- вопросы по безопасности входа (OTP/2FA).

## 4. Рекомендации по формулировке вопроса

- пишите кратко и по сути;
- указывайте тип транспорта и маршрут;
- уточняйте, на каком шаге возникла проблема.

## 5. Ошибки и fallback-сценарии

Если бот не дал корректный ответ:
1. переформулируйте вопрос;
2. проверьте стабильность интернет-соединения;
3. повторите запрос через 1-2 минуты.

Если сервис недоступен, пользователю выводится сообщение об ошибке и рекомендация повторить попытку позже.
