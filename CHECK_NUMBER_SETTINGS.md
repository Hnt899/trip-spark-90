## Что проверить в личном кабинете МТС Exolve

### 1. Проверка привязки номера к приложению

**Шаги:**
1. Личный кабинет МТС Exolve → **"Приложения"**
2. Выберите ваше приложение (ID: `e56a88f4-812b-413f-a5d3-cf1995c8ab24`)
3. Найдите раздел **"Номера"** или **"Привязанные номера"**
4. Проверьте, привязан ли номер `+7 992 669-08-70` к приложению
5. Если нет - **привяжите номер к приложению**

**Важно:** Номер должен быть привязан к приложению, чтобы его можно было использовать для отправки SMS через API.

### 2. Проверка настроек номера

**Шаги:**
1. Личный кабинет МТС Exolve → **"Нумерация"** → **"Мои номера"**
2. Найдите номер `+7 992 669-08-70`
3. Откройте настройки номера
4. Проверьте:
   - ✅ **Статус:** должен быть "Активен" или "Активирован"
   - ✅ **Разрешения:** должна быть включена опция "Отправка SMS"
   - ✅ **Тип номера:** должен быть указан как "Для отправки SMS" или подобное
   - ✅ **Привязка к приложению:** номер должен быть привязан

### 3. Проверка прав API ключа

**Шаги:**
1. Личный кабинет МТС Exolve → **"Приложения"**
2. Выберите ваше приложение
3. Перейдите в раздел **"API ключи"** или **"Ключи"**
4. Найдите ваш API ключ (JWT токен)
5. Проверьте права:
   - ✅ **Отправка SMS** - должно быть включено
   - ✅ **Использование номера** - должно быть разрешено

### 4. Проверка истории сообщений

**Шаги:**
1. Личный кабинет МТС Exolve → **"Сообщения"** → **"История сообщений"**
2. Найдите сообщения с `message_id`:
   - `593002169481824943`
   - `593004366458260018`
3. Откройте детали сообщения
4. Проверьте:
   - **Статус доставки** - что именно написано?
   - **Причина ошибки** - есть ли описание ошибки?
   - **Детали** - дополнительная информация

## Возможные причины "Неуспешно"

### 1. Номер не привязан к приложению
**Симптомы:** Статус "Неуспешно", ошибка о недоступности номера
**Решение:** Привяжите номер к приложению в настройках

### 2. Номер не активирован для отправки SMS
**Симптомы:** Статус "Неуспешно", ошибка о правах доступа
**Решение:** Активируйте отправку SMS для номера в настройках

### 3. Недостаточно прав у API ключа
**Симптомы:** Статус "Неуспешно", ошибка авторизации
**Решение:** Проверьте права API ключа, создайте новый если нужно

### 4. Номер заблокирован или ограничен
**Симптомы:** Статус "Неуспешно", ошибка о блокировке
**Решение:** Проверьте статус номера, свяжитесь с поддержкой

### 5. Проблемы с оператором получателя
**Симптомы:** Статус "Неуспешно" только для определенных номеров
**Решение:** Попробуйте отправить на другой номер
